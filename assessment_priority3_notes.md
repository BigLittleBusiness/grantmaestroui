## Priority 3: Medium Impact, Medium Effort (Polish & Delight)

### 1. Enhance Task Management
- **Rationale:** The current task list is basic.
- **Implementation approach:** Add drag-and-drop reordering, priority flags (High/Med/Low), and visual indicators for overdue tasks (e.g. red text for past dates).

### 2. Refine the File Vault
- **Rationale:** The current file vault is a simple list.
- **Implementation approach:** Add file type icons (PDF, Excel, Word), file size indicators, and a drag-and-drop upload zone instead of just a standard `<input type="file">`.

### 3. Global Search
- **Rationale:** Users will eventually have hundreds of grants.
- **Implementation approach:** Implement a global search bar in the header that searches across Grant Titles, Task Descriptions, and Team Members simultaneously.

## Related strategy notes from section 3
- Keep table density compact; do not over-simplify tables with heavy padding.
- Improve keyboard navigation where practical.
- Bulk actions are mentioned as a future Excel-like behavior, but they are outside the Priority 3 table.

## Implementation scope for this session
Target the three Priority 3 items listed above:
1. Task management polish
2. File vault refinement
3. Global search in header
